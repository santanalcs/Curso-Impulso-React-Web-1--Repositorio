Bootcamp – Impulso React Web Developer

1º Desfio do Curso: Git/GitHub

Descrição: 
Criação e Versionamento, na prática, passo a passo, utilizando o Sistema de Gerenciamento e Versionamento de Projetos – Git, 
aliado ao Ambiente de Colaboração e Repositório de Dados – GitHub.

Documento foi criado e disponibilizado no repositório GitHub.
Passo 1 - Foi criada a pasta e inicializada com o comando > git init.
Passo 2 - Foi criado o repositório no GitHub.
Passo 3 - Informei o repositório remoto criado para a pasta local com o comando git remote add.
Passo 4 - Em seguida utilizei o comando > git add * < para registrar e informar ao Git as alterações processadas na pasta.
Passo 5 - Neste, utilizei o comando > git commit < tornamdo disponível todas as alterações realizadas na pasta/arquivo.
Passo 6 - Por fim, enviei todo o pacote processado no repositório local, com os passso 3, 4 e 5, para o repositório remoto
	  no GitHub com o comando > git push.

Os demais versionamentos de edição deste documento foram executados com os passos 4, 5 e 6, listados acima.
